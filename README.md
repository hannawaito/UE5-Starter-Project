# 🎮 UE5 Starter Project

## 📌 Overview
This project is built using Unreal Engine 5 as a starter tutorial
for learning and developing core gameplay systems.

It includes:
- Basic player movement
- Camera control
- Blueprint-based architecture
- Simple environment setup

---

## 🛠️ Tech Stack
- Engine: Unreal Engine 5
- Language:
  - Blueprints (primary)
  - C++ (optional)
- IDE: Visual Studio 

---

## 📂 Project Structure
```
Content/
├── Blueprints/        # Core gameplay logic (Player, GameMode, etc.)
├── Maps/              # Levels
├── Materials/         # Materials & shaders
├── Meshes/            # Static/Skeletal meshes
├── UI/                # Widgets (HUD, menus)
└── Input/             # Input mappings
```

---

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone <your-repo-url>
```

### 2. Open Project
- Open `.uproject` file with Unreal Engine 5

### 3. Generate Project Files (if using C++)
- Right click `.uproject` → Generate Visual Studio project files

### 4. Run
- Click **Play** inside UE5 editor
